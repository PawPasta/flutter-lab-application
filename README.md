# Flutter Lab

Repository gồm 3 project Flutter phục vụ bài lab môn PRM392.

## Danh sách project

1. `flutter_calculator`
- Ứng dụng máy tính đơn giản.
- Cho phép nhập 2 số và thực hiện 4 phép toán cơ bản: cộng, trừ, nhân, chia.

2. `flutter_crud`
- Ứng dụng quản lý công việc (Task Manager).
- Thực hiện CRUD với SQLite (`sqflite`): thêm, xem danh sách, sửa, xóa task.

3. `flutter_firebase`
- Ứng dụng đăng nhập Firebase.
- Tích hợp `firebase_auth` + `google_sign_in` để đăng nhập Google và chuyển sang màn hình Home.

## Cách chạy nhanh

Mỗi project chạy độc lập. Vào từng thư mục rồi chạy:

```bash
flutter pub get
flutter run
```
